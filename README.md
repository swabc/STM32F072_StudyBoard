# STM32F072_StudyBoard
Study board for STM32F072 (CAN, LIN available)

Interfaces:
- CAN x1
- LIN x1
- High speed USB serial convertor x1 (for FlashLoaderDemo)
- Relay x4
- Analog Input x4 (0~33V)
- Button x4 (WK1, WK2, Reset, BOOT0)
- SPI extensive
- I2C extensive

Power supplies:
- USB 5V
- Optional LIN 12V
